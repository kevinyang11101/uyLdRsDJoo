# Python中医文化学习系统 python1016

## 项目概述

本项目是一个基于Python语言的中医文化学习系统，采用前后端分离的架构设计，前端使用Vue.js框架，后端采用Django框架，数据库使用MySQL。系统旨在为管理员、教师和学生提供一个便捷的中医知识学习与管理平台。

## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

- 用户管理（管理员、教师、学生）
- 课程管理
- 资源共享
- 在线测试
- 互动讨论

## 系统角色

- 管理员：负责系统维护、用户管理和资源管理。
- 教师：负责课程发布、教学资源和作业布置。
- 学生：参与课程学习、资源下载和在线测试。

## 运行环境

- 服务器：推荐使用Linux操作系统。
- Python环境：Python 3.x版本。
- 前端环境：Node.js版本需高于14。

## 部署步骤

1. 准备Python和MySQL运行环境。
2. 安装项目所需依赖库和框架。
3. 部署后端Django服务。
4. 部署前端Vue.js项目。
5. 配置域名和服务器。

## 目录结构

```
- backend                # 后端代码
  - apps                 # 应用目录
    - admin              # 管理员模块
    - course             # 课程模块
    - ...
  - settings.py          # 配置文件
  - urls.py              # 路由配置
- frontend               # 前端代码
  - src                  # 源代码目录
    - assets             # 资源文件
    - components         # 组件
    - views              # 页面
    - ...
  - package.json         # 前端依赖配置
- db                     # 数据库相关
  - migrations           # 数据库迁移文件
- doc                    # 项目文档
```
